# 🦷 ToothCare — Dental Clinic Website

A fully responsive front-end website for a dental clinic, built with HTML, CSS, and Bootstrap 5.

---

## 📋 Overview

**ToothCare** is a multi-page static website designed for a dental clinic. It provides patients with information about services, doctors, pricing, and allows them to book appointments and contact the clinic.

---

## 🗂️ Pages

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Hero section, services overview, team, testimonials, and treatment results |
| About | `about.html` | Clinic story, stats (3,000+ patients, 24 years of experience) |
| Services | `services.html` | Detailed list of all dental services offered |
| Dentists | `dentist.html` | Profiles of the qualified dental team |
| Pricing | `pricing.html` | Service pricing plans and FAQ section |
| Blog | `blog.html` | Dental tips and news articles |
| Contact | `contact.html` | Contact form and clinic location details |
| Login | `login.html` | Patient login page |
| Register | `register.html` | New patient registration page |

---

## ✨ Features

- **Responsive design** — works on all screen sizes using Bootstrap 5 grid
- **Appointment booking** section on the homepage
- **Doctor profiles** with photos for 8 qualified dentists
- **Patient testimonials** with photos
- **Before & after treatment gallery** (Stories section)
- **Pricing plans** with FAQ accordion
- **Blog section** for dental health articles
- **Login & Registration** pages for patient accounts
- **Interactive contact form**

## 🚀 Getting Started

No build tools or dependencies needed — just open the files in a browser.

```bash
# Clone the repository
git clone <repo-url>

# Open in browser
open index.html
```

Or simply double-click `index.html` to launch locally.
